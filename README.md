# mobile-robotics-frontier-exploration

## Setup

1. Clone the repo anywhere:
```bash
   git clone <repo-url> && cd mobile-robotics-frontier-exploration
```

2. Build the workspace:
```bash
   cd ros2_ws && colcon build --symlink-install && cd ..
```

3. In every new terminal, source the environment:
```bash
   source open_ros_env.sh
```
